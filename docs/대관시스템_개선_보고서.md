# 서초여성가족플라자 대관 신청 시스템 개선(안)

---

## 1. 현행 시스템 현황 및 문제점

현재 대관 신청은 공식 홈페이지(seochowomen.kr)에서 **양식을 다운로드 → 수기 작성 → 스캔 → 기관 이메일로 접수**하는 방식으로 운영되고 있습니다.

| 구분 | 현행 방식 | 문제점 |
|------|----------|--------|
| **신청** | 양식 다운로드 후 수기 작성·스캔·이메일 발송 | 접수까지 다수의 단계, 작성 오류 빈번 |
| **일정 확인** | 전화 문의 또는 직접 방문 | 실시간 예약 현황 파악 불가 |
| **접수·처리** | 이메일 수신 → 수동 대장 관리 | 누락·중복 위험, 처리 현황 추적 어려움 |
| **결과 안내** | 개별 전화·이메일 회신 | 담당자 업무 부담 과중 |
| **통계·보고** | 수기 집계 | 시간 소요, 정확성 한계 |

## 2. 개선 시스템 주요 기능

### 2-1. 신청자 화면

| 기능 | 설명 |
|------|------|
| **공간 탐색** | 강의실(9개실)·E-스튜디오·우리동네 갤러리 3개 카테고리, 층별 공간 안내 |
| **실시간 예약 현황** | 캘린더 기반 가용 일시 즉시 확인, 정규수업·차단일정 자동 반영 |
| **온라인 신청** | 신청자 정보·일시·목적·장비 선택을 웹 폼으로 입력, 복수 회차 일괄 신청 지원 |
| **자동 접수 확인** | 신청 완료 즉시 이메일 자동 발송 (신청번호·공간·일시 포함) |
| **결과 조회** | 신청번호 또는 이메일로 승인/반려 결과·요금 내역 실시간 확인 |
| **내 신청 관리** | 이메일 인증 기반 신청 이력 조회, 취소 가능 건 직접 취소 |

### 2-2. 관리자 화면

| 기능 | 설명 |
|------|------|
| **신청 관리** | 카테고리·상태·기간·검색어 필터, 묶음/개별 보기 전환 |
| **승인·반려 처리** | 건별/일괄 처리, 할인 적용(사유 기록), 반려 사유 입력 |
| **이메일 자동 발송** | 승인·반려·취소 시 맞춤 이메일 자동 발송, 템플릿 편집 지원 |
| **캘린더 뷰** | 월별 예약·정규수업·차단일정 통합 조회 |
| **일정 차단** | 정규수업(요일·시간 반복), 시간 단위·일 단위 수동 차단 |
| **통계** | 월별 신청 건수·이용 일수·수입 자동 집계 |
| **엑셀 내보내기** | 필터 적용 목록 및 개별 신청서 양식 다운로드 |

## 3. 기대 효과

| 항목 | AS-IS (수기) | TO-BE (온라인) | 효과 |
|------|------------|--------------|------|
| **신청 소요 시간** | 양식 작성·스캔·발송 약 30분 | 웹 폼 입력 약 5분 | **83% 단축** |
| **일정 확인** | 전화 문의 필요 | 실시간 캘린더 확인 | **즉시 확인** |
| **접수·처리** | 이메일 확인 → 수동 기록 | 자동 접수·알림·이력 관리 | **누락 방지** |
| **결과 안내** | 개별 연락 | 이메일 자동 발송 | **업무 경감** |
| **통계·보고** | 수기 집계 | 자동 집계·엑셀 출력 | **정확성 향상** |

---

## 4. 시스템 구성

- **프론트엔드**: Next.js 기반 반응형 웹 (PC·모바일 대응)
- **데이터 저장**: Google Sheets API 연동 (별도 DB 서버 불필요, 기존 업무 환경 활용)
- **배포 환경**: Vercel 클라우드 (자동 배포, SSL 인증서 포함)
- **이메일**: SMTP 연동 자동 발송
- **보안**: 관리자 인증, 입력값 검증, 접근 제어, 속도 제한
